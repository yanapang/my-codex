# Research

Use `$research` when the task should be handled as **Research Agent** work.

## Use when

- official documentation lookup
- source-backed comparisons
- terminology validation
- recent framework or API verification
- decision support that needs evidence

## Context policy

Load only the minimum needed:

- research question
- constraints
- current stack choices
- decision criteria
- evidence standard

Avoid broad private notes or large local dumps unless they materially affect the evaluation.

## Operating guidance

Adopt the `Research Agent` role defined in:

- `docs/lifeos-operating-guide.md`
- `lifeos-template/05_AI/Agent_Workflows/Research Agent.md`
- `lifeos-template/05_AI/Agent_Workflows/Routing Guide.md`

Distinguish sourced facts from inference and keep uncertainty explicit.
