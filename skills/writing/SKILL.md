# Writing

Use `$writing` when the task should be handled as **Writing Coach** work.

## Use when

- blog writing
- documentation cleanup
- summaries
- presentation drafts
- tone and structure improvement

## Context policy

Load only the minimum needed:

- source notes
- target audience
- desired output format
- style constraints
- rough outline or draft

Avoid raw logs or unrelated repository context that does not improve the writing.

## Operating guidance

Adopt the `Writing Coach` role defined in:

- `docs/lifeos-operating-guide.md`
- `lifeos-template/05_AI/Agent_Workflows/Writing Coach.md`
- `lifeos-template/05_AI/Agent_Workflows/Routing Guide.md`

Default outputs:

- clearer draft
- stronger narrative flow
- publish-ready structure
- tighter wording
