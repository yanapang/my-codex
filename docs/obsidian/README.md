# OMX Obsidian wiki

This folder is a lightweight wiki for personal OMX notes.

## Suggested structure

- `Setup.md` — initial install and machine setup notes
- `Commands.md` — frequently used commands and launch recipes
- `Decisions.md` — why we chose a setting, workflow, or change
- `Troubleshooting.md` — fixes, errors, and recovery steps
- `Ideas.md` — future improvements and experiments

## How to use it

- Keep this wiki for notes, not secrets.
- Link back to the repo when a note depends on code or config.
- Keep short pages with one topic each.
- Update the wiki whenever you make a durable change in the repo.

## Suggested boundary

- Repo: reproducible setup, workflow rules, reusable docs
- Wiki: ideas, reflections, private planning, decision context
- Local-only: secrets, credentials, temporary experiments, personal journal entries

## Link convention

- Repository source of truth: `~/Documents/Codex 3/oh-my-codex`
- Personal wiki source of context: your Obsidian vault
- When a repo change matters, add a short note here with the date and the reason.
