# Setup

## Goal

Track the steps needed to make OMX usable on a new machine.

## Checklist

- Clone the repo.
- Install Node.js 20+.
- Run `npm ci`.
- Run `npm run setup`.
- Run `npm run doctor`.
- Run a smoke test from the same shell you will actually use.
